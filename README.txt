Práctica 1 de la asignatura Estructura de Datos y Algoritmos de la Universidad Francisco de Vitoria.

DESCRIPCIÓN:

Programa encargado de gestionar radares de una pequeña población. Se requiere una carga de los radares al inicio de programa,
la cual puede ser manual o automática (por ficheros). En dicha carga se indicarán los datos de los radares (ID, Velocidad
límite, Umbral 0-20%, Umbral 20-40%, >40%).

/* ASIGNACIÓN CARPETAS */

bin ==> Ejecutable (u otros binarios)
data ==> Ficheros y datos
source ==> Programas sin compilar
